Hermes API (1.0.0)

Download OpenAPI specification:

Canonical Hermes OpenAPI contract for Project LOGOS. See Project LOGOS spec: Table 2 in Section 3.4 (Hermes API endpoints).

Hermes is the stateless language & embedding utility providing:

  • Speech-to-text (STT)
  • Text-to-speech (TTS)
  • Simple NLP preprocessing
  • Text embedding generation
  • LLM gateway proxy via /llm

All endpoints are stateless and do not interact with the HCG directly.

LLM Gateway

Proxy chat/completion requests through the configured provider (OpenAI, local, or echo fallback).

Request Body schema: application/json
required
Any of
prompt
required
string

Shortcut for a single user message when messages is omitted.

Array of objects (LLMMessage)

Conversation history forwarded to the provider.

provider
string

Override the configured provider (e.g., openai, echo, local).

model
string

Provider-specific model identifier override.

temperature
number <float> [ 0 .. 2 ]
Default: 0.7

Sampling temperature forwarded to the provider.

max_tokens
integer >= 1

Optional maximum number of tokens to generate.

object

Additional metadata stored alongside the request.

Responses

Local development server

http://localhost:8080/llm
