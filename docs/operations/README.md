# Operations & Verification Docs

Use this folder for runbooks, verification checklists, scenario scripts, metrics, and observability instructions.

## Files & Subfolders
- [Phase 2 Verification Guide](PHASE2_VERIFY.md) — milestone acceptance checklist.
- [Metrics Ideas](METRICS_IDEAS.md) — backlog of telemetry/coverage metrics.
- `audits/` — infra/test audit summaries (e.g., hardcoded parameter sweep).
- `demos/` — walkthroughs + asset lists for demos (e.g., M4 pick-and-place).

Docs such as scenario scripts, demo capture notes, and observability guides should be migrated here from `docs/demo/`, `docs/phase2/`, etc.
