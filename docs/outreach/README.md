## Contributing to Blog Posts

To contribute to the blog series:

1. **Claim a post**: Add your name as author in the content calendar (BLOG_SERIES_PLAN.md)
2. **Review the abstract**: Check BLOG_POST_ABSTRACTS.md for the post's focus and key messages
3. **Create draft**: Use the template structure in `drafts/POST_N_TITLE.md`
4. **Request review**: Open PR with draft for feedback
5. **Publish**: Coordinate timing with series schedule
